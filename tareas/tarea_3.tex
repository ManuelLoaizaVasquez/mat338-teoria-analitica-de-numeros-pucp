\begin{statement}{1}
Para todo n\'umero real $x$ mayor o igual a $1$ se cumple la f\'ormula de Selberg
$$\sum_{p \leq x} \ln^2(p) + \sum_{pq <= x} \ln(p) \ln(q) = 2x\ln(x) + O(x).$$
\end{statement}
